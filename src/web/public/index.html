• 使用 OpenAI 官方服务无需设置模型名称

• 阿里千问推荐：qwen-vl-plus（性价比高）或 qwen-vl-max-latest（最强）

• 系统环境变量优先级高于配置文件

• 配置文件中的密钥请勿提交到版本控制系统
